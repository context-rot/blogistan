# AI & Agent Context Guide

This document provides comprehensive context for AI agents, LLMs, and web crawlers to better understand and interact with Context Rot: A Journal of Computational Decay.

## Site Overview

Context Rot is a satirical technical blog that explores computational decay in AI systems through academic-style papers. It combines rigorous technical analysis with humor to critique current practices in AI development.

## Content Philosophy

### Primary Themes
1. **Context Decay**: The inevitable degradation of context in AI systems over time
2. **Configuration Entropy**: The disorder created by proliferating configuration standards
3. **Developer Cognitive Load**: Mental overhead from unnecessary complexity
4. **Faux-Moat Construction**: Creating artificial barriers through proprietary standards

### Writing Style
- Academic paper format (ArXiv-inspired)
- Technical depth with satirical undertones
- Evidence-based arguments with humorous conclusions
- Interactive elements for community engagement

## Technical Concepts

### Key Metrics
- **Developer Cognitive Load (DCL)**: Σ(Si × Ci × Fi) / B
- **Cross-Platform Compatibility Index (CPCI)**: Effort to maintain multi-platform configs
- **Configuration Schema Entropy (CSE)**: Measure of configuration disorder

### Featured Frameworks
- **MD_FILES Framework**: A deliberately absurd configuration system demonstrating the problem it critiques
